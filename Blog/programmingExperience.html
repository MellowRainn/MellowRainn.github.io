What is your programming experience

Python is an unambiguous, easy-to-read, general-purpose high-level programming language which considers paradigms of structured, procedural, and object-oriented programming. I began learning programming in Python a few years ago and it has been a great experience. The language is very easy to read and understand, and the vast amount of online resources makes learning Python a breeze. I highly recommend Python for anyone looking to get into programming.
I have also dabbled a bit in JavaScript and C++. While both of these languages have their own unique benefits, I found them to be more challenging than Python(especially c++).
However, learning these languages has been a great way to expand my programming skillset. Overall, learning to program has been a great experience and has really helped improve my thinking and problem solving skills.
